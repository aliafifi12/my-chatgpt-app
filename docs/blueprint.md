# **App Name**: ChatFlow

## Core Features:

- Chat Interface: Simple chat interface allowing users to input prompts.
- AI Chat Response: Integrate with the OpenAI API to process user prompts and generate responses. Use a tool that chooses when to use certain resources.
- Chat History: Display chat history with clear differentiation between user prompts and AI responses.
- Subscription Details Page: Create a basic pricing page outlining the features included in each subscription tier.
- Subscription Management: Enable users to manage their subscription (upgrade, downgrade, cancel).
- Payment processing via PayPal: Collect payments for subscriptions via PayPal.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a trustworthy and intelligent feel.
- Background color: Light gray (#F5F5F5), creating a clean and focused environment.
- Accent color: Teal (#009688) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern look.
- Use minimalist icons from a consistent set for intuitive navigation and functionality.
- Implement a clean, user-friendly layout, with a single column conversation view, and left-aligned messages for easy readability.
- Subtle animations (e.g., loading indicators, message arrival) to provide smooth feedback to the user.
